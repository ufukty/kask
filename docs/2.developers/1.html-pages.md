# Html pages

Kask creates the website pages out of individual the Markdown and Html files. Except those inside a `.kask` folder.

## Page templates

Kask look out for page templates to put the rendered Markdown content into, or to render the Html based pages. See [Page templates](./1.templating/page-templates.md) section for more.

## Layout and styles

While Markdown based page layout and styles can be customized at section basis; Html based pages can be customized at page basis.
